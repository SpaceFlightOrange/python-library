
About
=====

The Urbanairship ``reach`` (formerly known as ``wallet``) library is a Python library for
using the `Urban Airship Reach <http://urbanairship.com/>`__ web service API.

Requirements
============

As of version 0.1.0, Python 2.7 is required.
